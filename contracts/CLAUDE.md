# `contracts/` — the generated, committed cross-language contract

The source of truth for the wire boundary, GENERATED from `topos-types` and committed + diff-reviewed.

- **`schemas/`** — one JSON-Schema per top-level wire type (the per-loop contract oracle). **Generated by
  `xtask` — do not hand-edit.** Regenerate with `cargo run -p xtask -- gen-schema`; the
  `gen-schema --check` drift gate asserts the committed files match the types.
- **`openapi/`** — the plane spec (generated for the Rust plane once its routes are annotated).
- **`fixtures/`** — golden `--json` (per verb), placement (per harness, byte-exact), signed-pointer
  vectors, and the digest golden vector (filled in as the project matures).

Why committed-and-generated: a hand-edited protocol crate drifts; a generated-and-reviewed artifact does
not. Any consumer in another language pins these by the exact commit.
